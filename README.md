# 📍 Displacement Journey Database System

## 📖 Project Overview

The **Displacement Journey Database System** is a relational database project designed to track displacement journeys, humanitarian needs, aid distributions, and return barriers for displaced families.

The project aims to organize and manage humanitarian data efficiently while supporting reporting, analysis, and aid coordination processes.

This database was designed using relational database concepts, ERD modeling, normalization principles, and SQL schema design.

---

# 🎯 Objectives

* Track displaced families and individuals
* Record displacement events and movement history
* Manage humanitarian aid distributions
* Store needs assessments and return barriers
* Support reporting and humanitarian analysis
* Design a scalable relational database structure
